GitBook Documentation
=============

这本书是一个 **GitBook** 的完整文档 (gitbook.com 和 toolchain) 。

### 开源的工具

GitBook 是一个能够创建非常漂亮图书的工具，基于 git 和 Markdown。它能够将你的书以多种格式进行生产，比如：: **PDF**, **ePub**, **mobi** 或者直接生成一个 **网站**。

它是开源的，也是完全免费的。源代码可以在[GitHub](https://github.com/GitbookIO/gitbook)上找到。

对Gitbook的的问题和建议请在这里提出——[github.com/GitbookIO/gitbook/issues](https://github.com/GitbookIO/gitbook/issues)

### GitBook.com

GitBook.com 是一个在线创建图书的平台 ([www.gitbook.com](https://www.gitbook.com))
